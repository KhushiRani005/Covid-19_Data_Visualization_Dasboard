# Covid-19_Data_Visualization_Dasboard
📌 Project Overview

The COVID-19 Data Visualization Dashboard is an interactive web application built using Python, Pandas, Plotly, and Streamlit. It helps users explore and analyze COVID-19 data through dynamic visualizations and country-wise statistics.

The dashboard provides insights into confirmed cases, deaths, recoveries, and trends across different countries and regions.

🚀 Features
Interactive COVID-19 data visualization
Country-wise data filtering
Time-series analysis of COVID-19 cases
Global statistics overview
Interactive charts and graphs using Plotly
User-friendly Streamlit interface
Real-time exploration of pandemic trends
🛠️ Technologies Used
Python
Pandas
NumPy
Plotly Express
Streamlit
